A lightweight Streamlit application to track job applications and store them in a well-formatted Excel file.  
Designed for fast data entry, minimal UI clutter, and local-first storage.


## Features

- Simple form-based UI for quick job application logging
- Captures:
  - Company Name
  - Job Description URL (saved as a clickable hyperlink)
  - Date & Time of application
  - Application status (Heard back or not)
- Automatically formats the Excel file:
  - Bold headers
  - Frozen header row
  - Consistent column widths
  - Clickable hyperlinks
  - Proper date-time formatting
- Focused, distraction-free interface (no data table shown in UI)
- macOS compatible
